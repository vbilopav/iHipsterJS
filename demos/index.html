iHipsterJS Demos Collection Index Page

Demo apps are divided into three main group - each of them containing different approaches and aspects of this framework to demostrate.

Also, each of them contains at least one index.html page as entry point html document with following link to latest build of the framework:

<script type="module" src="../../../src/ihjs/build/1.0.0/ihjs.js"></script>

There is also commented reference to development script above the script tag - which can be used for debugging and development purposes:

<!-- <script type="module" src="../../../src/ihjs/dev/ihjs.js"></script> -->

Demo applications are following:

Remote data demos

These are simplest examples that will simply display data from remote JSON endpoint ↗ - in simple UL/LI list.

Demos are designed to demonstrate power and capability of templating system based on built-in JavaScript ES6 template literals  ↗ system in various different ways.
There is also example on how to bundle entire app into single file with the rest of the framework.

Click here to see full list:  ↗

Simple SPA (Single Page Application) demos

Simple SPA application is small SPA (with client router) application consisting of two screens:

- Input screen with input field (with default value vbilopav, that can be changed in query string) and button Show github user data
Button will navigate to another screen and display GitHub user info data by username provided in the input and passed as view parameter.

- Screen that displays GitHub user info data by username provided in the input and passed as view parameter in HTML table with Back button at the bottom that uses browser navigation to get back to previous screen.

Demos will demonstrate different approaches with templates and modules. There is also TypeScript example included.
