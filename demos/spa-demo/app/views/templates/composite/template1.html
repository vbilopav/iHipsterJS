This composite template view!

Templates can be composed form other templates.
To import other templates use this.template.import helper.
Parametrized templates start with "template!" and they are functions that can receive parameters. Plain text templates start with "template!" and they return string instead of function.
Modules can be imported too.

View location: 

/app/views/templates/composite/template1.html

Full route definition: 
"/composite": {
    view: "template!views/templates/composite/template1.html", 
    data: {
        title: "Composite template",
        category: "templates"
    },
    paramsMap: (...params) => params // any 
}

template1.html panel
template1.html imports template2.html:

${this.template.import("template!views/templates/composite/template2.html")({class: "panel-primary", params: this})}

Also, template import can pass parameters to each other. This route can receive any parameter.
Try to change them manually in adress bar to see what happens...
