Template variables

Templates can have variable. To declare variable, use arrow function that doesn't return anything, and set it to this
For example:

${()=>{this.myVariable = "some_value"}} 
${()=>{this.myVariable = "some_value"}}

Note that expression above allows multilne for more complex scenarios. If arrow function returns anything it will be redndered into in template. Example for multilne expression: ${ () => { this.myVariable2 = "some_value_2"; this.myVariable3 = "some_value_3"; } } 

${
    () => {
        this.myVariable2 = "some_value_2";
        this.myVariable3 = "some_value_3";
    }
}

To retrieve variable, use arrow function again and return a value: 
${()=>this.myVariable}

myVariable=${()=>this.myVariable}
myVariable2=${()=>this.myVariable2}
myVariable3=${()=>this.myVariable3}

Except myVariable4, if you want to have variable value available immidietly - just enclose it in self-execute function: 
${(() => {this.myVariable4 = "some_value_4"})()}
${(() => {this.myVariable4 = "some_value_4"})()} myVariable4 is accesible without arrow function: 
${this.myVariable4}
myVariable4=${this.myVariable4}
