Example of html template that imports and executes composite template

Templates can contain parts (chunks, components) of templates string, template functions or template promises.
Everythng between template expressions will be ignored and expressions will be returned in array that can be destructed. See template /app/views/components/composed-template.html.

Usage example: 

${
    () => {
        let [part1, part2, part3] = this.template.import("composite!views/components/composed-template.html")(args);
        this.compontent1 = part1;
        this.compontent2 = part2(1, 2);
        // etc
    }
}


View location: 
/app/views/components/composite-template.html

This example contains two text templates, one function template and one promise template which is returned after one second. Because of that, this entire view is rendered after one second.

${ }
compontent1:
${() => this.compontent1} compontent2:
${() => this.compontent2} compontent3:
${() => this.compontent3("Simple component 3", "template function")}
${() => this.compontent4("Simple component 4", "template promise function")}
